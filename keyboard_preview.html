⚙️ Enable Glass Keyboard

  1. Go to Settings → System → Languages & input
  2. Tap On-screen keyboard → Manage keyboards
  3. Toggle Glass Keyboard ON
  4. Select it as your current keyboard

This preview simulates the experience. On a real device, Android handles the keyboard settings.
